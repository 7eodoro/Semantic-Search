# Semantic-Search
TL;DR: E-commerce search is broken because it speaks "database" while customers speak "human." By applying Marketing Myopia theory, I built a hybrid PoC that uses LLM Intent Translation and Vector Embeddings to turn vague queries into precise, deterministic results. The result? No more "Zero Result" pages and a search that actually understands the user.
